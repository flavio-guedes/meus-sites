Privacidade e segurança de dados — leia antes de começar

⚠️

Pesquisas de segurança identificaram que o Trae coleta dados de uso e pode transmitir conteúdo de arquivos para servidores da ByteDance, empresa chinesa. Não é necessariamente um problema — mas depende do que você está construindo.

🚫 Evite usar: código proprietário de clientes, dados sensíveis de usuários, projetos com NDA (acordo de confidencialidade)

Glossário — os termos que você vai ouvir

Definições diretas, sem enrolação

📖 Termos técnicos explicados
Vibe Code / Vibe Coding
Estilo de programar onde você descreve o que quer em texto e a IA escreve o código. Como "vibrar" com a ideia em vez de programar linha por linha.
LLM — Large Language Model
Modelo de linguagem grande. É o tipo de IA por trás do Claude e ChatGPT — treinado em bilhões de textos para entender e gerar linguagem.
RAG
Técnica que faz a IA buscar informações externas antes de responder. Como dar um Google pra IA antes de ela falar — ela busca, depois responde.
MCP — Model Context Protocol
"Tomadas" que conectam a IA a ferramentas externas como Figma, banco de dados e GitHub. Padroniza como AIs se comunicam com sistemas reais.
Builder Mode
Modo do Trae que constrói aplicações do zero a partir de descrição em texto. O modo mais poderoso da ferramenta para criar produtos novos.
VS Code
Editor de código mais popular do mundo, da Microsoft. Gratuito e open-source. O Trae é construído em cima dele, herdando todas as suas funcionalidades.
GitHub
Plataforma onde devs guardam e compartilham código. É como um Google Drive especializado para projetos de software, com histórico de versões.
Deploy / Publicar
Ato de colocar uma aplicação no ar para outras pessoas acessarem. De "só no meu computador" para "qualquer um pode ver na internet".
