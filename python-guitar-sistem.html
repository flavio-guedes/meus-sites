VARIÁVEIS
Variáveis
O container de informação mais básico do Python
conceito

Uma variável é um espaço nomeado na memória que guarda um valor enquanto o script estiver rodando. Você escolhe o nome, usa = e atribui o valor.

Pense no Notion de uma agência: cada campo de propriedade é uma variável. O campo "cliente" guarda "Nike Brasil". O campo "deadline" guarda "15/02". Python funciona exatamente igual — você nomeia e atribui.

No Tank-G, variáveis guardam o nome do preset, o modelo do amp, o nível de drive — tudo que compõe um som específico.

TIPOS
Tipos de Dados
Texto, número, decimal e sim/não — como os formatos de campo de uma planilha
conceito

Python diferencia o tipo da informação. É como a diferença entre um campo de texto livre e um campo numérico numa planilha — cada um aceita um formato específico.

str → texto: nome do preset, modelo do amp
int → inteiro: drive de 0 a 10, tempo em BPM
float → decimal: taxa de engajamento 4.7%
bool → verdadeiro/falso: pedal ativo ou bypass

Entender tipos evita erros: você não consegue somar texto com número, da mesma forma que não coloca CPF num campo de data no seu CRM.

F-STRING
f-strings e Operadores
Merge tags dinâmicas — como personalização em e-mail marketing
conceito

Uma f-string é um texto com variáveis embutidas automaticamente. Coloca f antes das aspas e usa {variavel} dentro. Python substitui pelo valor na hora.

É exatamente como merge tags em ferramentas de e-mail marketing: no Mailchimp você escreve *|FNAME|* e cada destinatário vê seu nome. Na f-string você escreve {cliente} e Python insere o valor.

Os operadores calculam: + soma, - subtrai, * multiplica, / divide. Use para calcular scores, índices e métricas dos presets — igual calcular CPM ou ROI de campanha.

LISTAS
Listas
Coleção ordenada — como uma setlist, pauta ou cadeia de pedais
conceito

Uma lista é uma sequência ordenada de itens dentro de colchetes [ ]. Cada item tem uma posição começando do zero — e você acessa, adiciona, remove ou percorre todos os itens.

Pense no seu calendário editorial: segunda = post motivacional (posição 0), terça = carrossel (posição 1), quarta = reels (posição 2). A ordem importa e você pode modificar a pauta.

No Tank-G, listas guardam a ordem da cadeia de pedais (compressor → overdrive → delay → reverb), os estilos disponíveis na biblioteca ou todos os nomes de presets para exibir num menu.

DICIONÁRIOS
Dicionários
Ficha de campanha com campos nomeados — acessa pelo nome, não pela posição
conceito

Dicionários usam pares chave: valor dentro de chaves { }. Em vez de acessar pelo número da posição (como nas listas), você acessa pelo nome do campo — muito mais legível.

É como um briefing de campanha completo: você não fala "o segundo item do documento", você fala "o campo objetivo" ou "o campo budget". Cada campo tem um nome único e um valor correspondente.

No Tank-G, cada preset será um dicionário: nome, estilo, amp, drive, reverb, delay — todos acessados diretamente pelo nome do campo, como preset["drive"].

IF / ELIF
Condições
Árvore de decisão — como uma régua de classificação de conteúdo por performance
conceito

Condições permitem que o código tome decisões baseado em critérios. if = se isso for verdade, execute X. elif = senão se for aquilo, execute Y. else = em qualquer outro caso, execute Z.

É exatamente como uma régua de classificação de posts por performance no seu relatório mensal: "se engajamento acima de 5% → viral; se entre 3% e 5% → bom; se abaixo → revisar estratégia". Você já pensa assim todo mês — agora é código.

No Tank-G: se drive menor que 3 → preset clean; se entre 4 e 6 → crunch; se maior que 7 → lead distorcido.

conceito

JSON
O Formato Universal de Dados
Como um .csv que qualquer software no mundo consegue abrir
conceito

JSON é o formato universal de troca de dados na web. Ele armazena dicionários e listas Python num arquivo de texto que qualquer software — Python, JavaScript, APIs, ferramentas de BI — consegue ler e escrever.

Pense num arquivo .csv que todo mundo abre: Excel, Google Sheets, Python, Power BI. JSON é o equivalente para dados estruturados mais complexos. É o idioma comum entre sistemas.

Você já sabe fazer dicionários. json.dump() transforma em arquivo no disco. json.load() lê o arquivo e devolve seu dicionário. Os seus presets ficam salvos permanentemente — mesmo fechando o Python.

Organização em Módulos
Separar responsabilidades — como a estrutura de pastas de uma agência bem organizada
conceito

Módulos são arquivos .py separados por responsabilidade. Em vez de um script gigante com tudo misturado, você divide: um arquivo para funções de preset, outro para a biblioteca, outro para a interface.
